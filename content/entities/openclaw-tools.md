# OpenClaw tools

## Tools là gì?

Tools là các năng lực mà agent có thể gọi để làm việc ngoài việc chỉ trả lời bằng text.

Ví dụ:
- đọc file
- sửa file
- chạy lệnh shell
- đọc memory
- gọi session khác
- tìm web
- xử lý image

## Vai trò

Tools biến agent từ một chat model thành một hệ thống có thể hành động.
