# OpenClaw config and security

## Chủ đề này là gì?

Đây là cụm kiến thức về cách OpenClaw được cấu hình và cách giữ cho setup an toàn khi chạy thật.

## Vì sao quan trọng?

Nếu không hiểu config và security thì rất dễ:
- bật sai quyền
- mở nhầm bề mặt truy cập
- cấu hình channel không an toàn
- để lộ credentials hoặc state nhạy cảm

## Phạm vi

Chủ đề này bao gồm:
- `openclaw.json`
- gateway config
- channel config
- agents config
- tool profile / tool allow-deny thinking
- security warnings và hardening cơ bản
