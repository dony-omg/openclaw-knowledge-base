# OpenClaw memory-wiki bridge mode

## Chủ đề này là gì?

Đây là chủ đề về cách `memory-wiki` kết nối với active memory thông qua `bridge mode`.

## Vì sao quan trọng?

Nếu `isolated` mode giúp wiki đứng riêng như một knowledge vault, thì `bridge mode` giúp wiki hút thêm dữ liệu công khai từ hệ memory đang hoạt động.
