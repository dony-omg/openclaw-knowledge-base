# OpenClaw agents config

## Chủ đề này là gì?

Đây là phần cấu hình liên quan đến agents trong OpenClaw.

## Vì sao quan trọng?

Nếu channels là cửa vào, sessions là ngữ cảnh, tools là tay chân, models là bộ não suy luận, thì agents config là nơi định nghĩa từng “con người số” trong hệ thống.

## Các phần thường liên quan

- `agents.defaults`
- `agents.list`
- workspace
- model per agent
- tools per agent
- sandbox per agent
- identity / persona files
