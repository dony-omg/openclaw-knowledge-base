# OpenClaw project

## OpenClaw là gì?

OpenClaw là một gateway self-hosted để kết nối các nền tảng chat với AI agent, tools, sessions, memory, routing và các workflow đa agent.

## Vai trò trong hệ thống học tập của Trung

Đối với Trung, OpenClaw hiện đóng vai trò:
- trợ lý AI chạy qua Telegram và WhatsApp
- nền tảng học về session, routing, spawn, multi-agent
- nền tảng để kết hợp với Obsidian thông qua memory-wiki
- hạ tầng để xây knowledge base cá nhân
