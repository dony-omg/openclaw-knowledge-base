# OpenClaw sessions study plan

## Mục tiêu

Hiểu sessions như lớp ngữ cảnh hội thoại của OpenClaw.

## Các câu hỏi nên học

- Session key được hình thành từ đâu?
- DM và group khác nhau thế nào ở mức session?
- Channel khác nhau thì session khác nhau thế nào?
- Session liên hệ với routing ra sao?
- Session liên hệ với memory ra sao?

## Bài tập thực hành

- xem `openclaw status` để quan sát sessions hiện tại
- so sánh session Telegram và WhatsApp
- gắn phần hiểu này với note về channels và routing
