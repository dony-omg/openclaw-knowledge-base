# OpenClaw memory-wiki bridge study plan

## Mục tiêu

Hiểu bridge mode như một bước nâng cấp của memory-wiki, không phải như một công tắc nên bật ngay.

## Thứ tự học nên đi

### 1. Nắm chắc isolated mode
- vault structure
- compile
- lint
- source / concept / synthesis

### 2. Hiểu active memory là gì
- session memory
- durable memory
- search / recall
- exported artifacts nếu có

### 3. Hiểu bridge mode nối hai lớp nào
- active memory layer
- compiled wiki layer

### 4. Đặt câu hỏi trước khi bật
- active memory hiện tại có artifact để bridge không?
- mình muốn tự viết knowledge hay muốn wiki hút thêm dữ liệu tự động?
- mình có chấp nhận thêm độ phức tạp không?

## Bài tập thực hành

- đọc lại `concepts/openclaw-memory-wiki.md`
- đọc note này
- mô tả bằng lời: isolated khác bridge ở đâu
- chỉ bật bridge khi Trung thấy rõ lý do dùng nó
