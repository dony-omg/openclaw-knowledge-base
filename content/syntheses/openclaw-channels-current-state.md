# OpenClaw channels current state

## Kênh hiện đang hoạt động

Hiện tại setup đang có ít nhất:
- Telegram
- WhatsApp

## Trung nên quan sát gì?

- direct chat trên Telegram tạo session ra sao
- direct chat trên WhatsApp tạo session ra sao
- mỗi channel có đặc điểm gì khác nhau
- allowlist, policy, account, peer ảnh hưởng ra sao

## Câu hỏi nên tự trả lời

- Một message đi từ Telegram vào OpenClaw thế nào?
- Gateway dùng gì để biết route vào session nào?
- Nếu thêm 1 channel mới thì cần config những gì?
