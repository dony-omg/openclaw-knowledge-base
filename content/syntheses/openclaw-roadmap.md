# OpenClaw learning roadmap

## Giai đoạn 1: Foundation

- Hiểu gateway, agent, session, routing
- Dùng được Obsidian vault với memory-wiki
- Biết compile/lint wiki
- Biết đọc knowledge base hiện tại

## Giai đoạn 2: Knowledge workflow

- Tạo source note từ docs hoặc trải nghiệm thật
- Tạo concept note từ điều học được
- Tạo synthesis note để tổng hợp
- Cập nhật entities khi hiểu rõ hơn về hệ thống

## Giai đoạn 3: Advanced OpenClaw

- bridge mode cho memory-wiki
- multi-agent routing thực tế
- ACP / coding harness
- automation, tasks, heartbeats

## Giai đoạn 4: Ứng dụng vào dự án riêng

- dùng OpenClaw như personal assistant
- dùng OpenClaw như knowledge engine
- dùng OpenClaw như hệ thống làm việc gắn với dự án cá nhân
