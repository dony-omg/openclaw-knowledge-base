# OpenClaw tools study plan

## Mục tiêu

Hiểu tools như một hệ năng lực của agent, không chỉ là danh sách chức năng rời rạc.

## Các cụm nên học

### 1. File tools
- read
- write
- edit
- apply_patch

### 2. Shell tools
- exec
- process

### 3. Session tools
- sessions_spawn
- sessions_send
- sessions_list
- sessions_history
- session_status

### 4. Memory tools
- memory_search
- memory_get

### 5. Web/media tools
- web_fetch
- image

## Bài tập thực hành

- quan sát lúc agent đọc/ghi note trong Obsidian
- quan sát lúc agent chạy `openclaw wiki compile`
- hiểu tại sao một số việc cần exec, còn một số việc chỉ cần file tools
- phân biệt tool nào là internal, tool nào là external-facing
