# OpenClaw models and providers current state

## Setup hiện tại của Trung

Từ những gì đã thấy trước đó, setup hiện tại đang dùng model mặc định kiểu `openai-codex/gpt-5.4`.

Điều đó cho thấy Trung đang dùng một model khá mạnh cho:
- chat
- tool use
- giải thích kỹ thuật
- hỗ trợ thiết lập OpenClaw

## Ý nghĩa thực tế

Điểm mạnh của cách này là:
- chất lượng tốt
- không cần suy nghĩ quá nhiều về phân vai model ngay từ đầu
- phù hợp để học nền tảng OpenClaw

## Điều nên học tiếp

- model hiện tại được cấu hình ở đâu
- nếu có nhiều agent thì tách model theo vai trò như thế nào
- fallback/failover hoạt động ra sao
- khi nào nên đổi provider hoặc model
