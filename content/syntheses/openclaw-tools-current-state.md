# OpenClaw tools current state

## Trung đã dùng những gì?

Trong quá trình học vừa rồi, Trung đã gián tiếp dùng khá nhiều tool qua agent, ví dụ:
- đọc file trong workspace và vault
- viết note vào Obsidian vault
- sửa cấu trúc note
- chạy compile/lint cho memory-wiki
- kiểm tra trạng thái OpenClaw
- đọc báo cáo lint

## Ý nghĩa thực tế

Điều này cho thấy OpenClaw không chỉ là nơi để hỏi đáp, mà là hệ thống có thể thao tác thật trên tri thức, file và workflow.

## Điều nên học tiếp

- file tools khác nhau ra sao
- khi nào dùng exec, khi nào không
- session tools dùng để làm gì
- memory tools khác wiki tools thế nào
