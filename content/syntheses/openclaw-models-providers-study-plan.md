# OpenClaw models and providers study plan

## Mục tiêu

Hiểu model/provider như nền tảng quyết định chất lượng và hành vi thực tế của agent.

## Thứ tự học nên đi

### 1. Học phần khái niệm
- model là gì
- provider là gì
- vì sao phải tách hai khái niệm này

### 2. Học phần config
- model mặc định đang nằm ở đâu trong config
- per-agent model khác nhau cấu hình thế nào

### 3. Học phần vận hành
- tốc độ
- chi phí
- reliability
- fallback/failover

### 4. Học phần thiết kế hệ thống
- agent nào nên dùng model nào
- khi nào nên giữ một model cho mọi thứ
- khi nào nên tách model theo vai trò

## Bài tập thực hành

- xem model hiện tại trong `openclaw status`
- xem config model mặc định
- mô tả nếu có `main` + `coding` thì nên chia model ra sao
